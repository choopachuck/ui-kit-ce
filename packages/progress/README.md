# ProgressBar

Описание компонента

## API

| Свойство  | Тип                  | Описание                    |
| --------- | -------------------- | --------------------------- |
| `classes` | `Classes`            | `JSS-классы для стилизации` |
| `size`    | `'sm' ⎮ 'md' ⎮ 'lg'` | `Размер`                    |

## Пример использования

```javascript
import { ProgressBar } from '@v-uik/progress'
```
