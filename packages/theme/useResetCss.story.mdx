# useResetCss

Хук `useResetCss` позволяет сбрасывать стили браузера по умолчанию

Данный хук глобально объявляет единый стиль для всего документа, убирая лишние отступы и указывая корректные типы отображения, шрифты и внешний вид у HTML-элементов

## import

```tsx
import { useResetCss } from '@v-uik/base'
```

или

```tsx
import { useResetCss } from '@v-uik/theme'
```
